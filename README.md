# Ruboty::GoogleCalendar

Google Calendar API Wrapper for ruboty

## Installation

Add this line to your application's Gemfile:

    gem 'ruboty-google_calendar'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-google_calendar
